
CI-PHP - PHP Continuous Integration assistant
=============================================


Introduction
------------

CI-PHP is a tool made to assist with continuous integration workflows.


Basic usage
-----------

CI-PHP can be included in the project through composer:

```bash
# To install run:
composer require luka8088/ci

# To invoke run:
./vendor/bin/ci
```
